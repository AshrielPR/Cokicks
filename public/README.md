# CoKicks Public Assets

Esta carpeta contiene archivos publicos que Vite sirve desde la raiz del sitio.

## Logo

Coloca el logo real en:

```txt
public/brand/
```

Ejemplo recomendado:

```txt
public/brand/cokicks-logo.png
```

## Productos

Coloca fotos reales de listings en:

```txt
public/products/
```

Luego usa rutas como:

```ts
"/products/jordan-4-bred-1.jpg"
```

## Hero

Coloca fotos destacadas de Home en:

```txt
public/hero/
```
